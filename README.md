Service Call Analysis

This project is a completed analysis of the transcription of a HVAC Service Call. The analysis is hosted live at https://servicecallanalysis-1.onrender.com/. 

The website itself contains the full call transcript, with sections labeled according to what part of the call it is. Addtionally, the website contains banners on the right side that hold the different analysis sections.

Usage:

To see a detailed analysis of each section. Click on a chevron on one of the right banners for the section you are interested in.
